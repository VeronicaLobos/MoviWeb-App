About this Project

Description

The MovieWeb App is a web application that allows users to manage a list of movies, including adding, updating, and deleting movies. The app is designed to be user-friendly and easy to navigate, with a simple and intuitive interface. The app uses Flask and SQLAlchemy to manage the database, and it uses SQLite as the database. The app also fetches movie data from the OMDb API using the omdb_data_fetcher module (requires an API key).

This program is a learning project for the Software Engineering Bootcamp at MasterSchool. It showcases what I have learned about Python programming, API and web development. A step-by-step guide is included to help you understand how the code has been implemented.

Features

  • CRUD operations for managing movies, users and ratings.
  • The API is built using Flask and SQLAlchemy, and it uses SQLite as the database.
  • Endpoints for adding, updating, and deleting movies, adding users, and retrieving movie details.
  • Input validation in templates to ensure that the user provides valid data before submitting the form.
  • The API also fetches movie data from the OMDb API using the omdb_data_fetcher module (requires an API key).
  • SSR (Server-Side Rendering) is used to render HTML jinja2 templates for the web application, and a stylesheet (style.css) with a responsive minimalist design is included.
  • The app is designed to be user-friendly and easy to navigate, with a simple and intuitive interface.
  • DAL (Data Access Layer) is implemented using SQLAlchemy ORM with SQLite, through the DataManagerSQLite class, for managing database operations.
  • OOP (Object-Oriented Programming) is used to define the User and Movie classes, which represent the database tables.
  • The app is designed to be modular and easy to extend with additional features in the future.
  • Logging is implemented to track operations where a user ID is involved, such as adding or deleting movies.
  • Deletion confirmation for movies and users to prevent accidental deletions.

User Guide

These are the steps to run the MovieWeb App locally.

1. Setting Up the Environment

  • Set up a virtual environment (the app uses Python 3.9): 
    python -m venv venv
  • Activate the virtual environment: 
    # On Windows
    venv\Scripts\activate
    # On macOS/Linux
    source venv/bin/activate
  • Install the required packages with pip or your preferred package manager: 
    pip install -r requirements.txt

2. Clone the Repository

git clone https://github.com/VeronicaLobos/MoviWeb-App.git
cd MovieWeb-App

3. Set Up the Environment Variables

The app uses an API key to fetch movie data from the OMDb API.

You will need to register at https://www.omdbapi.com to obtain one.

Create a .env file in the root directory of the project and add the API Key to the .env file like this:

my_api_key="YOUR_OMDB_API_KEY"

Without this key, the app will not be able to fetch movie data from the OMDb API.

4. Install Dependencies

Make sure you have the required dependencies installed.

Use pip or your preferred package manager:

pip install -r requirements.txt

5. Run the Application

  • To run the CSR version of the app: 
    python app_ssr.py
  • To run the SSR version of the app: 
    python app_csr.py

6. Access the Application

Open your web browser and go to http://127.0.0.1:5002/home
